# ONG CHADIA — Site web institutionnel

Site vitrine de l'ONG CHADIA (Tchad), construit avec Next.js 15 + TypeScript + Tailwind CSS.

## Commandes de base

```bash
npm run dev      # Démarrer le serveur de développement (localhost:3000)
npm run build    # Générer le build de production
npm run start    # Démarrer le serveur de production (après build)
npm run lint     # Vérifier le code avec ESLint
```

## Documentation

- `docs/prd.md` — Product Requirements Document (22 stories, 4 epics)
- `docs/architecture.md` — Architecture technique (diagrammes)
- `docs/brainstorming-session-results.md` — Session de brainstorming initiale

## Stack technique

- **Framework** : Next.js 15 (App Router, SSG)
- **Langage** : TypeScript strict
- **Styles** : Tailwind CSS
- **Hébergement** : Vercel
- **Email** : Resend
